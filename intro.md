# MBA 590: Advanced AI Strategy

## Prompting and Agentic Frameworks - Spring 2026

Welcome to **MBA 590: Advanced AI Strategy**, a comprehensive course designed to equip leaders and strategists with the knowledge and skills to effectively leverage sophisticated language models and emerging agentic systems.

---

## Course Overview

This advanced course provides an in-depth exploration of:

- **Prompt Engineering**: From fundamentals to advanced techniques (Chain-of-Thought, RAG, Self-Refine)
- **LLM Evaluation**: Rigorous frameworks for assessing quality, safety, and business suitability
- **Agentic Systems**: Architecture, capabilities, and strategic applications
- **Implementation Strategy**: Tech-ready operating models, governance, and ethical frameworks
- **Strategic Leadership**: Technology strategy formulation, ROI measurement, and future trends

---

## Course Structure

The course consists of **15 weeks** organized into four parts:

### Part I: Prompt Engineering Mastery (Weeks 1-4)
Build a strong foundation in prompt engineering, from basic principles to advanced techniques like few-shot learning, chain-of-thought reasoning, and retrieval-augmented generation.

### Part II: Evaluation & Agentic Systems (Weeks 5-8)
Learn to evaluate LLM outputs rigorously and explore the emerging world of agentic AI systems, including multi-agent architectures and business applications.

### Part III: Implementation Strategy (Weeks 9-12)
Develop practical skills for implementing AI in organizations, including operating model design, governance frameworks, and ethical considerations.

### Part IV: Strategic Leadership (Weeks 13-15)
Master strategic thinking for AI initiatives, including technology strategy formulation, ROI measurement, and preparing for future trends.

## How to Use This Site

### Weekly Notebooks
Each week features an interactive Jupyter notebook containing:
- Learning objectives
- Conceptual explanations
- Python code examples
- Hands-on exercises
- Discussion questions
- Key takeaways

### Assignments
Four major assignments allow you to apply course concepts:
1. **Prompt Engineering** - Master advanced prompting techniques
2. **Agentic Framework** - Design and implement agentic systems
3. **Operating Model** - Develop AI-ready organizational structures
4. **Strategic Plan** - Create comprehensive AI strategies

### Running the Code
You can run the notebooks in several ways:
- **Google Colab**: Click the Colab button on any notebook page
- **Local Installation**: Clone the repository and run with Jupyter
- **Docker**: Use the provided Docker configuration for a reproducible environment
