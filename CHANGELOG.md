# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-18

### Added
- Complete set of 15 weekly Jupyter notebooks covering the full curriculum
- Assignment templates for all 4 major course assignments
- Sample datasets for practical exercises (CSV and JSON formats)
- Prompt templates library with reusable business templates
- Python utilities module with helpers for LLMs, data, prompts, and evaluation
- Comprehensive README with course overview and setup instructions
- Requirements.txt with pinned dependencies
- Environment.yml for Conda users
- .env.example template for API key configuration
- .gitignore for Python/Jupyter projects
- CONTRIBUTING.md with contribution guidelines
- FAQ.md with common questions and solutions
- TROUBLESHOOTING.md with setup and runtime issue solutions
- Dockerfile and docker-compose.yml for containerized environment
- GitHub Actions CI/CD workflow for automated testing
- Notebook testing script (scripts/test_notebooks.py)
- Export scripts for PDF, HTML, and slides conversion
- Documentation for all modules and utilities

### Course Content
- Week 1-4: Prompt Engineering (foundations through RAG)
- Week 5: LLM Evaluation metrics and frameworks
- Week 6-8: Agentic Systems and Multi-Agent Architectures
- Week 9-10: Tech-Ready Operating Models
- Week 11-12: Governance and Ethics
- Week 13: Technology Strategy and Portfolio Management
- Week 14: ROI Measurement and Financial Analysis
- Week 15: Future Trends and Strategic Leadership

### Infrastructure
- Modular Python utilities for common tasks
- Reusable prompt templates organized by category
- Sample datasets for hands-on practice
- Automated testing framework
- Docker support for reproducible environments
- CI/CD pipeline for quality assurance

## Version History

### Version Numbering
- **Major version (X.0.0)**: Significant course restructuring or new content modules
- **Minor version (1.X.0)**: New notebooks, major feature additions, or substantial updates
- **Patch version (1.0.X)**: Bug fixes, minor improvements, content corrections

### Release Schedule
- Major releases: Annually (before each academic year)
- Minor releases: As needed (new content, significant updates)
- Patch releases: As needed (fixes and improvements)
